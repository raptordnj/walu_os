# Security Policy

## Supported Versions

The `main` branch is the only actively supported line.
Tagged releases receive best-effort security fixes.

## Reporting a Vulnerability

Do not open public issues for suspected vulnerabilities.

Use one of these channels:
- GitHub private vulnerability report (Security tab)
- Direct contact with repository owner: `@raptordnj`

Include:
- affected commit/tag
- reproduction steps
- impact and attack surface
- proposed mitigation (if available)

You will receive an acknowledgement within 72 hours.
